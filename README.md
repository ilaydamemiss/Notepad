# Basit Notepad (C#)

Bu proje, C# ile yapılmış basit bir Notepad uygulamasıdır. Temel olarak metin yazabilir, açabilir ve kaydedebilirsiniz.
## Özellikler
- Yeni dosya oluşturma  
- Var olan dosyayı açma  
- Dosya kaydetme  
- Metin düzenleme: kopyala, kes, yapıştır

## Gereksinimler
- Windows işletim sistemi  
- Visual Studio veya başka bir C# geliştirme ortamı

## Kullanım
1. Projeyi indirin veya klasörü açın.  
2. Visual Studio ile projeyi açın.  
3. Çalıştırarak Notepad’i kullanabilirsiniz.
